STEP 5: (OPTIONAL) INSTALL LIVE SERVER
─────────────────────────────────────────────────────────────────────

This makes changes appear instantly in your browser!

1. Click Extensions icon (left sidebar - looks like 4 squares)
2. Type "live server" in search box
3. Find "Live Server" by Ritwick Dey
4. Click "Install"
5. Wait for installation to complete
6. Click "Reload" if it appears

After installing:
- Right-click index.html
- Click "Open with Live Server"
- Your website opens in browser!
- Changes update automatically!

HOW TO EDIT EACH FILE
─────────────────────────────────────────────────────────────────────

EDITING index.html (CONTENT & TEXT):
─────────────────────

1. Click on index.html in left panel
2. You'll see HTML code on the right
3. Use Ctrl+F to find text (like "Every Cakes")
4. Click on the text and edit it
5. Press Ctrl+S to save
6. Website updates! ✓

EDITING style.css (COLORS & DESIGN):
─────────────────────

1. Click on style.css in left panel
2. You'll see CSS code on the right
3. Find colors near the top (like #a78167)
4. Click and change the color code
5. Press Ctrl+S to save
6. Website colors update! ✓

EDITING script.js (FEATURES):
─────────────────────

Don't touch this unless you know JavaScript!
The code is already perfect. ✓

QUICK EDITING TIPS
─────────────────────────────────────────────────────────────────────

FIND TEXT:
Ctrl+F → Type what you're looking for

REPLACE TEXT:
Ctrl+H → Find and Replace

SAVE:
Ctrl+S

UNDO (Oops!):
Ctrl+Z

GO TO LINE:
Ctrl+G → Type line number → Enter

SELECT ALL:
Ctrl+A

COPY:
Ctrl+C

PASTE:
Ctrl+V

COMMENT/UNCOMMENT:
Ctrl+/ (useful for testing)

COMMON EDITING TASKS
─────────────────────────────────────────────────────────────────────

TASK 1: Change Business Name
FILE: index.html
SEARCH FOR: "Every Cakes"
CHANGE TO: "Your Name"

TASK 2: Change Main Heading
FILE: index.html
SEARCH FOR: "Handcrafted Cakes for Every Occasion"
CHANGE TO: "Your heading"

TASK 3: Change Main Color
FILE: style.css
SEARCH FOR: "--warm-brown: #a78167;"
CHANGE TO: "--warm-brown: #YOUR_COLOR_CODE;"
(Find color codes at colorpicker.com)

TASK 4: Add Your Photos
PUT: Your cake photos in the CakeWebsite folder
NAME: cake-hero.jpg, cake1.jpg, cake2.jpg, etc.
WEBSITE: Automatically shows them!

TASK 5: Change Contact Email
FILE: index.html
SEARCH FOR: "hello@everycakes.com"
CHANGE TO: "your@email.com"

SIDE-BY-SIDE EDITING (Advanced)
─────────────────────────────────────────────────────────────────────

Want to see two files at once?

1. Click on first file (like index.html)
2. Hold Ctrl and click on second file (like style.css)
3. Now you see both at the same time!
4. Edit both together!

TESTING CHANGES
─────────────────────────────────────────────────────────────────────

After you edit:

1. Press Ctrl+S to save
2. Look at your browser (if using Live Server)
3. It updates automatically!

HOW TO ADD YOUR PHOTOS
─────────────────────────────────────────────────────────────────────

In your CakeWebsite folder, you need these images:
- cake-hero.jpg (main image at top)
- cake1.jpg, cake2.jpg, cake3.jpg (showcase cards)
- birthday-cake.jpg, wedding-cake.jpg, dessert-cake.jpg (offers)

How to add them:

1. Download or take photos of your cakes
2. Save them as JPG files
3. Name them EXACTLY as above (case matters!)
4. Drag them into the CakeWebsite folder
5. Done! The website shows them automatically!

No code changes needed! ✓

COLOR CODES - WHERE TO FIND THEM
─────────────────────────────────────────────────────────────────────

1. Go to: https://colorpicker.com
2. Click a color or search "pink", "blue", etc.
3. Copy the HEX code (like #ff6b9d)
4. Go to style.css
5. Search for: "--warm-brown: #"
6. Replace the color code
7. Save (Ctrl+S)
8. Color changes! ✓
